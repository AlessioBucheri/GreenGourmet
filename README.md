# The Project

Last project for start2Impact React course.
The object of this course is to build a complete and full working webapp with the framework React.
The goal is a webapp where a user can search and filter the best veggie recipe from Spoonacular API by ingredient.
You cand find my work here:
https://greengourmet.netlify.app/
